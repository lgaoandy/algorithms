**Object**

_Characteristics_

- unordered collection of key-value pairs
- keys must be string or symbol data type
- values can be any data type
- not iterable

_Actions_

- insert - O(1)
- remove - O(1)
- access - O(1)
- search - O(n)

_Methods_

- Object.keys() - O(n)
- Object.values() - O(n)
- Object.entries() - O(n)
